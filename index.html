Yihong 字体规范包说明

本字体规范包用于一鸿艺术项目输出，包括图文、PDF、PPT、图表、网页等。字体使用规范如下：

一、中文字体规范

  • 中文标题：微软雅黑加粗（msyhbd.ttc）
  • 中文副标题：微软雅黑（msyh.ttc）
  • 中文正文：思源宋体（SourceHanSerifSC-Regular.ttf）

二、英文字体规范

  • 英文字体：Times New Roman（Regular/Bold/Italic）

三、字体示例

【标题示例】这是中文标题文字（微软雅黑加粗）

【副标题示例】这是副标题文字（微软雅黑）

【正文示例】思源宋体用于中文正文排版，具备良好的阅读性与跨平台兼容性。

[English Sample] This is body text in Times New Roman Regular.

四、引用方法（HTML 页面内嵌）

<style>
@font-face {
    font-family: 'SourceHanSerifSC';
    src: url('fonts/SourceHanSerifSC-Regular.ttf') format('truetype');
}
@font-face {
    font-family: 'msyh';
    src: url('fonts/msyh.ttc') format('truetype');
}
@font-face {
    font-family: 'msyhbd';
    src: url('fonts/msyhbd.ttc') format('truetype');
}
@font-face {
    font-family: 'Times New Roman';
    src: url('fonts/TimesNewRoman-Regular.ttf') format('truetype');
}
</style>

五、使用建议

  • 所有 PDF 输出需嵌入字体，确保跨平台显示一致性
  • 所有 PPT 制作需手动设置各级标题与正文字体
  • 图表可使用微软雅黑加粗作为图例与标题，思源宋体作为辅助信息

如需更新字体，请重新打包字体目录，并修改 index.html 引用路径。
